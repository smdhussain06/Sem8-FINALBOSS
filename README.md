# 📚 AU Arrear Portal - Engineering Physics (PH3151)

> **Target: All Clear.** A comprehensive, exam-oriented study portal designed to help Anna University students master Engineering Physics (Regulation 2021) and clear their arrears with distinction.

[View the Portal Here](https://smdhussain06.github.io/Sem8-FINALBOSS/)

## ✨ Key Features

### 📖 100% Past Paper Coverage
Contains every unique question extracted from the 4 most recent Anna University past papers:
- **April/May 2024**
- **Nov/Dec 2023**
- **April/May 2023**
- **Nov/Dec 2022**

### ✍️ Official AU Framework Answers
All Part B (13/16 Marks) and Part C (15 Marks) answers are structured according to the "Anna University Strategy" to maximize mark potential:
- **Introduction**: Precise definitions and context.
- **Diagrams**: High-quality educational illustrations for every major concept.
- **Derivations**: Cleanly rendered formulas using **KaTeX**.
- **Comparisons**: Detailed tables for differentiating similar topics.
- **Applications & Pros/Cons**: Real-world context and critical analysis.
- **Conclusion**: Formal summaries for a professional finish.

### 📱 Mobile-First Experience
- **Fully Responsive**: Study on the go via smartphone, tablet, or desktop.
- **Math Scroll Support**: High-complexity formulas feature horizontal scrolling to ensure readability on small screens.
- **Concept Mastery**: Every technical answer includes an "Easy Learning" analogy to build intuition before memorization.

## 🛠️ Technology Stack
- **Frontend**: Vanilla HTML5, CSS3 (Mobile-first), Javascript (ES6+)
- **Mathematics**: KaTeX (for high-performance formula rendering)
- **Design**: Inter & Google Fonts, Vibrant Dark/Light Mode aesthetics

## 📂 Project Structure
- `index.html`: Main application entry point.
- `style.css`: Mobile-optimized design system.
- `app.js`: Core rendering logic and KaTeX integration.
- `ph3151_questions.js`: The central question database.
- `data.js`: Subject metadata and curriculum info.
- `*.png`: Educational diagrams and extracted paper pages.
